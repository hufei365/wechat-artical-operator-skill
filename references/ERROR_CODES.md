# 错误码参考与输出格式

## 错误码完整列表

| 错误码 | 含义 | 处理方案 |
|--------|------|----------|
| 40001 | access_token 无效/过期 | 自动清除缓存后重试 |
| 40003 | 无效 AppID | 检查 WECHAT_APP_ID 配置 |
| 40125 | AppSecret 错误 | 检查 WECHAT_APP_SECRET 配置 |
| 40163 | media_id 失效 | 重新上传封面图 |
| 40164 | IP 不在白名单 | 在公众号后台「设置与开发→基本配置」添加服务器 IP |
| 45009 | 接口调用超限 | 等待次日重试 |

## 失败响应格式

```
❌ 操作失败：{错误描述}

错误码：{errcode}
建议：{解决方案}
```

## 发布状态码

| 状态码 | 含义 |
|--------|------|
| 0 | 发布成功 |
| 1 | 发布中 |
| 2 | 原创失败 |
| 3 | 常规失败 |
| 4 | 平台拒绝 |
| 5 | 成功但审核不通过 |
