# 微信公众号主题样式预览

本项目参考 doocs/md 等主流微信编辑器，提供 **5 套精美主题** 样式。

---

## 🎨 主题列表

### 1. Doocs 绿（默认）

**特点：** 简洁清新，适合技术文章、干货分享

**主色：** `#07C160`（微信绿）

**适用场景：**
- 技术博客
- 教程文档
- 开源项目宣传

**示例效果：**
- 标题：绿色下划线/左边框
- 引用：浅绿色背景
- 代码：深色背景

---

### 2. 科技蓝

**特点：** 专业稳重，适合企业宣传、产品介绍

**主色：** `#1a73e8`（Google 蓝）

**适用场景：**
- 企业新闻
- 产品发布
- 行业报告

**示例效果：**
- 标题：蓝色渐变背景
- 表格：蓝色边框和表头
- 链接：蓝色下划线

---

### 3. 简约黑

**特点：** 极简主义，适合文字类、观点类文章

**主色：** `#000000`

**适用场景：**
- 随笔杂文
- 观点评论
- 文学创作

**示例效果：**
- 标题：纯黑居中，字间距加宽
- 分割线：纯黑实线
- 整体：黑白灰三色

---

### 4. 文艺橙

**特点：** 温暖活泼，适合生活类、情感类文章

**主色：** `#ff6b35`（活力橙）

**适用场景：**
- 生活分享
- 情感故事
- 美食旅游

**示例效果：**
- 标题：橙色圆角背景
- 引用：橙色左边框 + 浅橙背景
- 分割线：橙色虚线

---

### 5. 商务灰

**特点：** 专业正式，适合商业报告、行业资讯

**主色：** `#4a5568`（高级灰）

**适用场景：**
- 商业分析
- 行业资讯
- 金融报告

**示例效果：**
- 标题：灰色边框
- 表格：灰色表头
- 整体：低饱和度配色

## 🚀 使用方法

### 命令行

```bash
# 列出所有主题
python3 scripts/md_to_wechat.py --list-themes

# 使用指定主题转换文件
python3 scripts/md_to_wechat.py \
  --file article.md \
  --output out.html \
  --theme tech_blue

# 使用指定主题转换文本
python3 scripts/md_to_wechat.py \
  --text "# 标题\n\n正文" \
  --theme warm_orange
```

### 在 Skill 中指定主题

告诉 AI 你想要的主题：

```
帮我把这篇文章发到公众号，用科技蓝主题
```

或

```
用简约黑主题排版这篇文章
```

## 💡 自定义主题

如果需要自定义主题，编辑 `scripts/themes.py`，添加新的主题配置：

```python
THEME_CUSTOM = {
    "name": "custom",
    "display_name": "自定义主题",
    "primary_color": "#your_color",
    "styles": {
        "h1": "你的 h1 样式",
        "h2": "你的 h2 样式",
        # ... 其他样式
    },
}

# 添加到 THEMES 字典
THEMES["custom"] = THEME_CUSTOM
```
